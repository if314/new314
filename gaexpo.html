Georgia Expo

Due to the success of the redesign of the Custom Exhibit Backdrops homepage, Management decided to task me with redesigning the printed fabrics section of our parent site GeorgiaExpo.com.

Problem

Since the original release of the printed fabric category page, we have seen very little customer interest. Bounce rates are over 75%. How might we update the category page to reduce bounce rates?

Goal

Raise traffic and reduce bounce rates.
Solution
  • Update the main menu to have a link in SHOP BY FABRIC for printed fabric, not only in SHOP.
  • Update the images to have a clear, clean, cohesive look.
  • Add a thin gray border around the category images to appear visually organized.
  • Shorten the product names. The category page name is displayed directly on top of the product name witch creates repetition.
  • Removed the rent a backdrop, research shows users are not interested in this service.

User Reaserch

We performed 20 user tests to better understand navigation to the Printed Fabrics page. We asked the participants to perform three simple tasks and provide verbal feedback while working through them.

  • Navigate to the Printed Fabrics category
  • Find the Printed Fabric Table Runner (Any Size)
  • Order a 28in x 80in table runner
Results The majority of users try looking for "Printed Fabrics" in the drop-down menu item "Shop By Fabric".

User Interviews

From user interviews, we gathered information, took suggestions. Trying to understand the pain points the user is facing while navigating the printed fabrics category page.

“From first glance, I can't tell which picture goes with which product there are too many flying parts.”

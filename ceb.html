Custom Exhibit Backdrops

ExhibitBackdrops.com

Exhibit Backdrops provides large format printed graphics for tradeshows and events. We analyzed and optimized the website for smooth browsing, ease of navigation, and faster checkout process.

Problem

How might we redesign ceb's website to help the user understand our product line and quickly find the printed fabric solution they need.

Goal

To be the go-to source that tradeshow goers of every experience level, budget and show rely on to complete their printed fabric display
Solution

Simplify the navigation menu to represent the product line better. Highlight design services and our tradeshow themed blog

Defind the user Survey existing customers, test usability with new users, Create user persona, explore stress cases. To do that we broke it down to these tree steps.

Copy Use plain language, short sentences, and paragraphs simple words, bullet points, and straight forward direction. Providing explicit content, direct information, eliminating fluff, and confusion.

Products Simplify products offered taking the guesswork out, Ease to use forms, clear instructions, add an economy line. Provide design templates easy to use by non-designers.

User Reaserch

We performed 20 user tests to better understand navigation to the Printed Fabrics page. We asked the participants to perform three simple tasks and provide verbal feedback while working through them.

  • Navigate to the Printed Fabrics category
  • Find the Printed Fabric Table Runner (Any Size)
  • Order a 28in x 80in table runner
Results The majority of users try looking for "Printed Fabrics" in the drop-down menu item "Shop By Fabric".
